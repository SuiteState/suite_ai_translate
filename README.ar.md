# ترجمة Discuss بالذكاء الاصطناعي (Discuss AI Translate)

**ترجمة الرسائل بالذكاء الاصطناعي لـ Odoo Discuss و WhatsApp و
سجلّات المحادثة (chatter).**

تواصَل بسهولة عبر اللغات — ترجم الرسائل الواردة عند الطلب،
واكتب الرسائل الصادرة بلغتك ثم أرسِلها بلغة المستلم مع معاينة
فورية قبل الإرسال.

---

## المزايا

- **ترجمة الرسائل الواردة عند الطلب**
  افتح قائمة النقاط الثلاث في أي رسالة واختر *AI Translate*،
  ثم حدّد اللغة المستهدفة. يظهر النص المترجم أسفل فقاعة
  الرسالة مباشرة مع إبقاء النص الأصلي سليمًا.

- **اكتب بلغتك، أرسِل بلغتهم**
  فعِّل وضع الترجمة في مربع الكتابة، واكتب بالعربية أو الصينية
  أو أي لغة مدعومة، لتَرى معاينة فورية بالإنجليزية (أو أي لغة
  مستهدفة) أثناء الكتابة. اضغط **Use** لإرسال النسخة المترجمة.
  يظهر رابط "View original" على الرسالة المُرسَلة لاستعادة النص
  الأصلي الذي كتبتَه.

- **يعمل في كل أماكن Discuss**
  القنوات الداخلية، محادثات WhatsApp، سجلّ المحادثة في السجلّات،
  الرسائل المباشرة، الدردشة الحيّة (live chat).

- **تكامل أصلي مع الذكاء الاصطناعي في Odoo**
  يستخدم إطار عمل الذكاء الاصطناعي المدمج في Odoo 19 — لا حاجة
  لخدمات خارجية إضافية. أضِف مفتاح API من **OpenAI** أو
  **Google Gemini** وتكون جاهزًا. إذا ضبطتَ الاثنين، يتم التبديل
  التلقائي عند فشل أحدهما.

- **صديق للتكاليف**
  تُخزَّن الترجمات لكل رسالة عبر جدول Odoo الأصلي
  `mail.message.translation`، فلا تتكرر استدعاءات API عند
  عرض نفس الرسالة مرارًا.

- **صديق للخصوصية**
  لا يغادر أي شيء نسختك من Odoo عدا نص الرسالة المطلوب ترجمته،
  ويُرسَل مباشرة إلى مزوّد الذكاء الاصطناعي الذي ضبطتَه.

## المتطلبات

- Odoo 19 Enterprise
- وحدتا `ai` و `ai_app` الأصليتان (مضمّنتان في Enterprise)
- مفتاح API من **OpenAI** أو **Google Gemini** مُعرَّف في
  *الإعدادات → AI*

---

## التثبيت والإعداد

1. ثبِّت الوحدة.
2. انتقل إلى **الإعدادات → AI**، وأضِف مفتاح API لمزوّد واحد على الأقل.
3. في الصفحة نفسها، اختر نموذج الترجمة الافتراضي (الافتراضي
   `gpt-5-mini` — سريع واقتصادي).

هذا كل شيء. يظهر إجراء *AI Translate* على كل رسالة مباشرة،
ويحصل مربع الكتابة على مفتاح تبديل الترجمة.

### إعدادات متقدّمة (اختيارية)

مَعاملات نادرة الاستخدام تُضبَط عبر *إعدادات النظام*
(*التقني → المَعاملات → مَعاملات النظام*) دون واجهة مستخدم،
والقيم الافتراضية تُغطّي 99% من الحالات:

| المَعامل                                   | الافتراضي | الغرض                              |
|--------------------------------------------|-----------|------------------------------------|
| `suite_ai_translate.max_input_chars`        | `8000`    | الحد الأقصى لطول الرسالة بالأحرف   |
| `suite_ai_translate.temperature`            | `0.2`     | درجة عشوائية أخذ العينات من النموذج |
